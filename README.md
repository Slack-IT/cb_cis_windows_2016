# cb_cis_windows_2016

Chef Cookbook for applying CIS Benchmark settings to Windows 2016 Systems.

## Requirements

### Platform:
* Windows 2016

### Chef
* Chef 12.?

### Cookbooks
* windows community cookbook (~> 4.0.0)
* windows-security-policy community cookbook (~> 0.3.7)
